Installation
------------

1.  Clone the repository:

    `git clone https://github.com/twoos123/asadali-portfolio.git`

2.  Navigate to the project directory:

    `cd asadali-portfolio`

3.  Install dependencies:

    `npm install`

4.  Start the development server:

    `npm start`
